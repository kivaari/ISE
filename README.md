## Представление проекта

### Название проекта
**Cloudberries**

### Идея проекта
Создание веб-платформы, которая соединяет владельцев простаивающего серверного оборудования (или мощных ПК) с пользователями, которым нужны вычислительные мощности для различных задач:
- Рендеринг видео и 3D-графики
- Обучение нейронных сетей и ML-моделей
- Хостинг веб-приложений и сайтов
- Научные вычисления и симуляции

**Ценность:**
- Для **поставщиков** — возможность монетизировать простаивающее оборудование
- Для **клиентов** — доступ к вычислительным мощностям по цене ниже, чем у крупных облачных провайдеров (AWS, Google Cloud, Azure)

## Границы и Scope проекта

### Бизнес-цели
1. Создать удобный и интуитивный интерфейс для поиска и аренды серверов
2. Автоматизировать процесс оплаты и учета времени аренды
3. Обеспечить безопасность сделок между арендатором и владельцем оборудования
4. Минимизировать комиссию платформы для привлечения пользователей

### Функциональные задачи

#### MVP (Минимально жизнеспособный продукт):
1. **Система регистрации и авторизации**
  - Два типа пользователей: «Поставщик» (Host) и «Клиент» (Guest)
  - Верификация аккаунтов
  
2. **Каталог серверов**
  - Отображение характеристик: CPU, RAM, GPU, SSD, цена за различное время использования
  - Фильтрация и поиск по параметрам
  - Рейтинг поставщиков
  
3. **Система бронирования**
  - Выбор периода аренды
  - Калькулятор стоимости
  - Корзина и оформление заказа
  
4. **Биллинг и платежи**
  - Интеграция платежной системы
  - История транзакций
  - Автоматическое списание за продление
  
5. **Личные кабинеты**
  - **Для клиента:** управление арендованными мощностями, статистика использования
  - **Для поставщика:** добавление серверов, статистика доходов, управление доступностью
